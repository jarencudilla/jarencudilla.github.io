# jarencudilla.github.io

Personal hub and portfolio for Jaren Cudilla — builder of things on the internet since 2007.

This page replaces `jaren.cudilla.net` (RIP). GitHub Pages does the same job for free, with no renewal anxiety.

---

## What this is

A launchpad. Not a blog, not a portfolio in the traditional sense. It's a single page that presents the CTRL+ALT+SURVIVE network — six independent sites built and maintained by one person — alongside open work on GitHub.

The six sites are the product. This page just routes people to them.
